Colour Clock Configuration File Documentation
This file is usually generated automatically. It is advised you leave it alone. Colour Clock comes with no warranty at all and what ever happens to it is all your own fault.


Colour Clock Configuration File Located At:
%appdata%\ColourClock\config.xml


Configuration Values


<Colour1>colour</Colour1>
ARGB Colour Value for Shape 1
  • colour: Colour Value


<Size1>x,y,width,height</Size1>
Size Value for Shape 1
  • x: Shape location from left edge of window
  • y: Shape location from top edge of window
  • width: Shape width
  • height: Shape height


<Shape1>shape</Shape1>
Shape Value for Shape 1
  • shape: Shape of Shape1. Can be
    • Square
    • Rectangle
    • OutlinedSquare
    • OutlinedRectangle
    • Elipse
    • Circle
    • OutlinedElipse
    • OutlinedCircle

<DisplayString>windowTitle</DisplayString>
Window display title
  • windowTitle: Title of the window. Can contain
    • %24time: The current time in 24hr format.
    • %time: The current time in 12hr format.
    • %ampm: Display am/pm.


<UpdateIcon>enabled</UpdateIcon>
Update icon to current time
  • enabled: Update the icon to the current time. Can be
    • true or 1: Icon changes with the time.
    • false or 0: Icon remains the same.


<Style>windowStyle</Style>
Update icon to current time
  • windowStyle: The frame of the window. Can be
    • borderless: Default no border window.
    • windowed: Traditional Windows window based frame.
    • simple: A simplified version of windowed.
    • tool: A toolbar style window. Does not show in taskbar.
    • other: Hidden values for experimentation and debugging.


<Size0>x,y,width,height</Size0>
Size of current window and location
  • x: Window location from left edge of screen
  • y: Window location from top edge of screen
  • width: Window width (window border inclusive)
  • height: Window height (window border inclusive)


<Background>backgroundColour</Background>
ARGB Colour Value for the Window Background
  • backgroundColour: Colour Value


<Outline>outlineThickness</Outline>
Sets the thickness for the window outline. Note this is different from the window frame, see Style.
  • outlineThickness: Thickness of border around edge of window. Accepts [0-9]*.


<OutlineColour>outlineColour</OutlineColour>
ARGB Colour Value for the Outline. See Outline
  • outlineColour: Colour Value


<AlwaysOnTop>alwaysOnTop</AlwaysOnTop>
Make the window always visible
  • alwaysOnTop: Sets if the window is always visible. Can be
    • true or 1: Window is always visible.
    • false or 0: Normal.
